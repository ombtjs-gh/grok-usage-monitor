# Grok Monitor - 複数アカウント使用量モニター

Grok (xAI) の複数アカウントの使用量・レート制限・リセット日時をデスクトップ上で常時監視するオーバーレイ型モニターアプリです。

## 機能

- **常に最前面表示**: 他のウィンドウを操作しても隠れないオーバーレイ表示
- **透明度調整**: 10%〜100% の範囲でリアルタイムに透明度を変更可能
- **複数アカウント対応**: 複数の Grok アカウントを追加・管理
- **OAuth2 ログイン**: `auth.x.ai` を利用した安全な認証（PKCE 対応）
- **使用量表示**: 残クエリ数/トークン数、使用率バー、リセットまでのカウントダウン
- **自動更新**: 設定間隔で使用量を自動ポーリング
- **クロスプラットフォーム**: Windows, macOS, Linux に対応

## 技術スタック

- **フレームワーク**: Tauri 2.x (Rust + React/TypeScript)
- **認証**: OAuth2 with PKCE
- **ストレージ**: ローカル暗号化ファイル (将来的に Keychain/Credential Manager 対応予定)

## インストール

### 前提条件

- Node.js 18+
- Rust 1.70+
- OS 別の Tauri 依存関係 ([Tauri prerequisites](https://tauri.app/start/prerequisites/))

### ビルド

```bash
# 依存関係のインストール
npm install

# 開発モードで実行
npm run tauri dev

# プロダクションビルド
npm run tauri build
```

## 使い方

1. アプリを起動
2. 「＋ アカウント追加」をクリックして OAuth2 ログイン
3. ブラウザで xAI アカウントにログインし、許可
4. アカウントが追加され、使用量が自動的に表示されます

## 設定

- **透明度**: スライダーでウィンドウの透明度を調整
- **更新間隔**: デフォルト 30 秒でポーリング（設定変更可能）
- **テーマ**: ダーク/ライトモード（システム設定に連動）

## セキュリティ

- トークンはローカルで暗号化して保存
- 平文でのトークン保存は行わない
- OAuth2 PKCE フローによる安全な認証

## 注意事項

⚠️ **重要**: 本アプリは以下の点にご注意ください：

1. **非公式 API の利用**: 一部機能は grok.com の内部 API を使用しています。xAI 側の変更により動作しなくなる可能性があります。
2. **利用規約**: 個人利用を想定しています。過度なポーリングや複数アカウントの意図的な制限回避は xAI の ToS に抵触する可能性があります。
3. **レート制限**: ポーリングが使用量を消費しないよう、適切な間隔を設定してください。
